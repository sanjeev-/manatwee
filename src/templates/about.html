my goal is to make a simple, clutter-free, and completely independent site for film & television sentiment aggregation & analysis

all twitter sentiment and movie reviews are scraped automatically and i use natural language processing and sentiment analysis to determine whether the review is positive or negative (via textblob)

the backend is in django, front end is bootstrap, and machine learning / analysis is done using the python ML toolkit (sklearn,nltk,pandas,etc)
